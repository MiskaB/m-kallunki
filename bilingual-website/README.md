# Bilingual Website Project

This project is a bilingual website that supports both English and Finnish languages. It is designed to be adaptive for both mobile and laptop devices, ensuring a seamless user experience across different screen sizes.

## Project Structure

The project consists of the following files and directories:

```
bilingual-website
├── en
│   └── index.html          # English version of the homepage
├── fi
│   └── index.html          # Finnish version of the homepage
├── assets
│   ├── css
│   │   └── styles.css      # CSS styles for the website
│   ├── js
│   │   └── scripts.js      # JavaScript for interactive elements
│   └── fonts               # Directory for custom fonts
├── .gitignore              # Files and directories to ignore by Git
├── README.md               # Documentation for the project
└── CNAME                   # Custom domain configuration for GitHub Pages
```

## Setup Instructions

1. **Clone the Repository**: 
   Clone this repository to your local machine using the following command:
   ```
   git clone https://github.com/yourusername/bilingual-website.git
   ```

2. **Navigate to the Project Directory**: 
   Change into the project directory:
   ```
   cd bilingual-website
   ```

3. **Open the Project**: 
   Open the project in your preferred code editor.

4. **Run the Website**: 
   You can open the `index.html` files in the `en` and `fi` directories in your web browser to view the English and Finnish versions of the homepage, respectively.

## Project Goals

- To provide a user-friendly bilingual website that caters to both English and Finnish speakers.
- To ensure the website is responsive and works well on various devices.
- To implement interactive features using JavaScript for a better user experience.
